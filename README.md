# Section Extractor

A small frontend utility for extracting sections, CSS class names, and component groups from HTML/CSS files.

I built this tool to help me inspect project files faster while working on frontend layouts, sidebar structures, and reusable UI sections.

## Features

- Extract section names from HTML
- Detect CSS class names
- Group layout-related classes
- Preview extracted output
- Copy or download the result
- Light and dark theme support

## Why I built this

When working on frontend projects, I often need to check which sections and class names are used across the layout.  
This tool helps me quickly scan and organize those names without manually searching through files.

## Tech Stack

- HTML
- CSS
- JavaScript

## Usage

1. Open the app.
2. Upload or paste your HTML/CSS content.
3. Select the section or class group you want to inspect.
4. Copy or download the extracted output.

## Project Status

This is a personal frontend utility tool.  
It is still improving, but it is already useful for checking project structure and CSS class naming.
